PLATFORMS

Uptime Client currently compiles & runs on these platforms:
- Linux
- *BSD (FreeBSD, OpenBSD & NetBSD confirmed)
- UnixWare
- Ultrix
- Solaris
- Irix


INSTALLATION

Installation requires these three, hopefully easy, steps:

1. edit config.h
2. for Solaris: remove the comment before "OPTS = -lsocket -lnsl"
3. type "make"
4. run upclient


PATCHES

We welcome any patches send in by you. But we would like you to
ask to code in our style. We don't use any spaces in code aligning,
only tabs. Just take a look at our code and you'll know how we prefer
the code :-)
